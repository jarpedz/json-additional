## JSON Additional

## Problem
ปัญหาที่ฉันเจอในงานของฉัน ฉันมีหน้า deploy version webapp โดยสิ่งที่อาจจะผิดบ่อยที่สุด จะเป็นใน ของ config.json โดย developer จะมีลง git ชื่อไฟล์ว่า config-prd.json 
โดยหน้าที่ของฉัน จะต้องทั้งสองไฟล์ มาเทียบกันว่ามี config ในบ้างที่เพิ่มมา

## Main Idea 
ฉันจึงต้องการโปรแกรมสำหรับเทียบ json เหล่านี้ โดยให้แบ่งเป็น 2 ฝั่งเอาไว้สำหรับเทียบ แล้วสามารถ import การเพิ่มเติมได้

วิธีการใช้งาน
ให้ copy ข้อมูลจาก .json ทั้ง 2 ฝั่ง แล้วทำการเทียบ และ import ส่วนที่เพิ่มเติมเข้าไป ข้อมูลหลัก

## ความต้องการ
ให้ทำตาม main idea มาก่อน 
ในส่วนของ หลังบ้าน จะมีการ login เพื่อใช้งานเท่านั้น ไม่เก็บข้อมูล json ใดๆ

## stack 
ฉันสร้าง framework ไว้แล้วเป็น react โดยใช้ supabase เป็น ฐานข้อมูล หาก ใช้ auth โดยใช้ zustand เพื่อจัดการ state ได้ก่อนก็ได้
